# Retrospective & Learning

## Overview

The Retrospective & Learning phase closes the loop on each cycle. AAVA uses AI to synthesize retrospective feedback, track action items, and identify patterns for continuous improvement.

This phase covers three core flows:

1. **Retrospective Synthesis** — AI-powered analysis of retro feedback
2. **Action Item Tracking** — Ensuring follow-through on improvement commitments
3. **Continuous Improvement** — Identifying trends and systemic improvements

## Inputs

- Team retrospective notes
- Previous action items
- Sprint metrics and outcomes

## Outputs

- Synthesized retro themes
- Prioritized action items
- Improvement trend analysis
