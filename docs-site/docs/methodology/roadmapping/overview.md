# Roadmapping

## Overview

The Roadmapping phase translates prioritized features into a time-bound plan. AAVA helps generate roadmaps that account for dependencies, capacity, and strategic sequencing.

This phase covers three core flows:

1. **Roadmap Generation** — Creating timeline-based roadmaps from prioritized backlogs
2. **Dependency Analysis** — Identifying and managing cross-team dependencies
3. **Timeline Estimation** — Estimating delivery timelines with confidence ranges

## Inputs

- Prioritized feature list
- Team capacity data
- Technical dependency information

## Outputs

- Visual roadmap with milestones
- Dependency graph
- Timeline estimates with risk factors
