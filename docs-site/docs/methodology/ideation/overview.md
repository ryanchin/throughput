# Ideation & Prioritization

## Overview

The Ideation & Prioritization phase transforms research insights into ranked opportunities. AAVA uses structured scoring frameworks accelerated by AI analysis.

This phase covers three core flows:

1. **Opportunity Scoring** — Quantifying the value and effort of each opportunity
2. **Feature Prioritization** — Ranking features using weighted criteria
3. **Impact Mapping** — Visualizing how features connect to business goals

## Inputs

- Research synthesis outputs
- Validated OKRs
- Technical feasibility assessments

## Outputs

- Scored opportunity backlog
- Prioritized feature list
- Impact maps linking features to outcomes
