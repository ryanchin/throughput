# Code Chat

An AI-powered conversational agent that lets you ask questions about your codebase in plain English — and get cited answers grounded in your actual source code.

## Questions It Answers

- "How does the authentication flow work in this codebase?"
- "Where is the payment processing logic, and what does it do?"
- "What API endpoints exist for user management?"
- "How are database migrations structured in this project?"

## Example Use Cases

- A new developer joins the team and needs to understand how the existing codebase handles authentication, authorization, and session management — without spending days reading code.
- A PM wants to understand what a feature does under the hood to write better acceptance criteria for an enhancement.
- A tech lead reviewing a pull request wants to quickly understand how a module works before providing feedback.

## What Makes It Special

- Answers are grounded in your actual code — every response includes file references and line citations
- Conversational interface — ask follow-up questions naturally
- Understands code across languages and frameworks
- No setup required beyond connecting your repository
