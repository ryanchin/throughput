# Feasibility

Evaluate whether a feature is buildable, then build the financial case to justify the investment. These two flows work together — import feasibility estimates directly into a business case.

## Flows in This Category

- **Feasibility Analyzer** — Technical feasibility with effort estimates, timelines, and Go/No-Go recommendations
- **Business Case Builder** — Financial analysis with NPV, IRR, ROI, and sensitivity analysis
