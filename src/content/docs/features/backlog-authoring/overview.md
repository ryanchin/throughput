# Backlog Authoring

Generate professional product artifacts — PRDs, epics, features, and user stories — with knowledge base context, citations, and consistent formatting across your organization.

## Flows in This Category

- **PRD Generator** — Comprehensive 10-section Product Requirements Documents with citations and professional exports
- **Epic Creator** — Generate epics with features and stories decomposed from a single input
- **Feature Creator** — Generate features with user stories and acceptance criteria
- **Story Creator** — Generate individual user stories with Gherkin acceptance criteria and story point estimates
