# Feature Creator

Generates a feature with decomposed user stories — each with structured acceptance criteria, business value, and story point estimates.

## When to Use Feature Creator

Use Feature Creator when you have a defined feature and need to break it into sprint-ready user stories. The output is: Feature → User Stories.

## Example Use Cases

- A team needs to decompose a large feature into individual user stories with acceptance criteria before sprint planning.
- A PM wants to quickly structure a feature request from stakeholders into implementable stories.

## What Makes It Special

- Import context from Ideation, Scope Definition, Goal sessions, PRDs, or parent Epics
- Gherkin-format acceptance criteria and Definition of Done checklists
- Story point estimates for sprint planning
- Push individual stories directly to Jira
