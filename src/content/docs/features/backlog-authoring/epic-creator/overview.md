# Epic Creator

Generates a complete epic with decomposed features and user stories — structured with acceptance criteria, business value, story point estimates, and a dependency graph.

## When to Use Epic Creator

Use Epic Creator when you're starting a new initiative and need to break a large body of work into features and stories. The output is a full hierarchy: Epic → Features → User Stories.

## Example Use Cases

- A PM has output from the Ideation Engine and needs to turn the top ideas into well-structured epics for the delivery teams.
- A product leader wants to decompose a strategic initiative into a complete epic with features and stories before PI planning.

## What Makes It Special

- Import context from Ideation, Scope Definition, Goal sessions, PRDs, or Competitive Analysis
- Generates the full hierarchy in one pass — epic, features, and stories
- Gherkin-format acceptance criteria and Definition of Done checklists
- Dependency graph showing what depends on what
- Knowledge base integration ensures generated artifacts reflect your organization's conventions
- Push directly to Jira when ready
