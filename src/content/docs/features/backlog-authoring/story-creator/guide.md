# How to Use the Story Creator

This guide walks you through generating a single, well-structured user story.

## Step 1: Provide Input

- **Title** (required) — Clear name for the story
- **Description** (required) — What the user needs and why

### Examples of Great Input

**Title:** "CSM Views Account Health Trend"

**Description:** "As a Customer Success Manager, I want to view a 90-day health score trend line for any account so that I can identify whether the account is improving or declining and prepare accordingly for the next check-in."

## Step 2: Review the Generated Story

The output includes:

- Acceptance criteria in Gherkin (Given/When/Then) format
- Definition of Done checklist
- Business value justification
- Story point estimate

## Step 3: Refine and Export

- Refine with AI or edit manually
- Copy to clipboard or push directly to Jira

## Tips

- Write descriptions in "As a [user], I want [goal] so that [reason]" format for best results
- The generated acceptance criteria are sprint-ready — review but don't over-edit
