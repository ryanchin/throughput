# Story Creator

Generates individual user stories with structured acceptance criteria in Gherkin format, business value, and story point estimates.

## When to Use Story Creator

Use Story Creator when you need a single, well-structured user story — for example, to fill a gap in an existing backlog or to quickly turn a request into a sprint-ready item.

## Example Use Cases

- A newly hired PM wants to learn the organization's artifact standards by seeing AI-generated examples that follow best practices.
- A PM needs to quickly turn a customer request into a well-structured user story for the next sprint.

## What Makes It Special

- Gherkin-format acceptance criteria (Given/When/Then)
- Definition of Done checklist
- Story point estimate
- Knowledge base integration for organizational conventions
- Push directly to Jira
