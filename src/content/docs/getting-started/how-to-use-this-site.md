# How to Use This Site

This documentation site is your reference guide for the AAVA methodology.

## Navigation

Use the **sidebar** on the left to browse by section. The documentation is organized into:

- **Getting Started** — Start here if you're new
- **AAVA Methodology** — Detailed guides for each lifecycle phase
- **Certifications** — Certification track information
- **Guides** — Practical how-tos

## Search

Use the **search bar** at the top to find specific topics. Try searching for terms like "goal extraction" or "sprint planning".

## Embedded Content

Some pages include embedded content like videos, slides, and diagrams. These are rendered inline for a seamless reading experience.

## Related Platform

This documentation complements the [Throughput training platform](https://throughput.aava.ai), where you can take structured courses and earn certifications.
