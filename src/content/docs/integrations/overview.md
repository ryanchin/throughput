# Integrations

Product Studio connects to the tools your team already uses, keeping your workflows in sync and eliminating duplicate data entry.

## Jira Cloud

Bi-directional sync with Jira Cloud. Push epics, features, and user stories directly to your Jira backlog. Pull real-time data for PI planning, progress tracking, and defect management.

### What You Can Do

- Push stories, features, and epics from the Story Generator directly to Jira
- Pull sprint data for Progress Tracker and Defect Manager
- Configure field mappings (story points, sprint, parent, team, priority, labels, components)
- Auto-detect field mappings or configure manually

### Setup

1. Connect via OAuth or Personal Access Token (PAT) for self-hosted Jira
2. Configure hierarchy mapping — map your epic, feature, and story types to Jira issue types
3. Set up field mappings — the system auto-detects common fields with confidence scores

## Azure DevOps

Full integration with Azure DevOps for teams on the Microsoft stack.

### What You Can Do

- Push work items from the Story Generator to Azure DevOps
- Pull sprint and work item data for Progress Tracker and Defect Manager
- Import teams from Area Paths

### Setup

1. Connect via OAuth
2. Select your project
3. Configure field mappings
